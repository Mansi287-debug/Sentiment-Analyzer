# Sentiment Analyzer

This project is a **Sentiment Analysis application built using Python and Natural Language Processing (NLP)**.  
It analyzes input text and determines whether the sentiment expressed is **Positive, Negative, or Neutral**.

---

## 📌 Project Overview
Sentiment analysis is a technique used in NLP to identify the emotional tone behind a piece of text.  
This project demonstrates how machine learning and NLP techniques can be used to classify text sentiment.

---

## 🚀 Features
- Analyze sentiment of text input
- Classifies sentiment as Positive, Negative, or Neutral
- Simple and easy-to-use Python implementation

---

## 🛠️ Technologies Used
- Python
- Natural Language Processing (NLP)
- NLTK / TextBlob (if used)
